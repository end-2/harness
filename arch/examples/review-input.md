# 아키텍처 리뷰 입력 예시

> design 에이전트 출력 + RE spec 산출물

---

이 파일의 입력은 다음 두 가지를 결합합니다:

1. **design 에이전트 출력**: `design-output.md` 참조
   - `architecture_decisions` (AD-001 ~ AD-005)
   - `component_structure` (COMP-001 ~ COMP-005)
   - `technology_stack` (8개 항목)

2. **RE spec 산출물**: `design-input.md` 참조
   - `requirements_spec` (FR-001 ~ FR-008, NFR-001 ~ NFR-003)
   - `constraints` (CON-001 ~ CON-005)
   - `quality_attribute_priorities` (usability, availability, security, maintainability)

리뷰 에이전트는 이 두 입력을 대조하여 설계의 적합성을 검증합니다.
