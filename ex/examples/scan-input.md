# 구조 스캔 입력 예시

> Task Manager API — Express + TypeScript 프로젝트 (경량 → 중량 경계 근처)

---

## 입력

```yaml
project_root: /workspace/task-manager-api
exclude_patterns: []
depth_override: null
```

### 프로젝트 설명

Express + TypeScript 기반의 태스크 관리 REST API 프로젝트. Prisma ORM으로 PostgreSQL을 사용하며, Jest로 테스트를 수행합니다. Docker로 컨테이너화되어 있고, GitHub Actions CI가 구성되어 있습니다.

파일 약 65개, 디렉토리 깊이 4, 언어 1개(TypeScript), 프레임워크 2개(Express, Prisma) → 중량 모드 예상.
